# SaldoCerto Web

Versao para navegador do app de gastos.

## Rodar no computador

```powershell
cd "C:\Users\Matheus T.I\Desktop\App iPhone Gastos\web"
python -m http.server 3000 --bind 0.0.0.0
```

## Abrir no iPhone

1. Conecte o computador e o iPhone no mesmo Wi-Fi.
2. Descubra o IP do computador.
3. Abra no Safari:

```text
http://IP-DO-COMPUTADOR:3000
```

Os dados ficam salvos no navegador em que voce usar o app.
