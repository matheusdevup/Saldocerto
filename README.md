SaldoCerto
App nativo para iPhone, feito em SwiftUI, para acompanhar gastos com base no salario.

O que o app faz
Permite informar salario mensal, custos fixos e meta de reserva.
Calcula quanto sobra para gastos variaveis no mes.
Mostra quanto voce pode gastar por dia.
Avisa quando o ritmo de gastos esta acima do saudavel.
Permite registrar despesas e acompanhar as ultimas movimentacoes.
Estrutura
SaldoCerto.xcodeproj: projeto para abrir no Xcode.
SaldoCerto/: codigo-fonte do app.
Como abrir
Copie a pasta para um Mac com Xcode.
Abra SaldoCerto.xcodeproj.
Escolha um simulador de iPhone.
Ajuste o time de assinatura em Signing & Capabilities, se o Xcode pedir.
Rode o app.
